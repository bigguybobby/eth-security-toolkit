# Ethereum Security Toolkit

Open-source smart contract security toolkit for the Ethereum ecosystem. Public good funded by the Ethereum Foundation ESP.

## Features
- Pre-deployment security scanner (static analysis + dynamic testing)
- Automated audit report generation
- Common vulnerability pattern detection
- Gas optimization suggestions
- Integration with Foundry/Hardhat workflows
- CI/CD security pipeline plugin

## Tech Stack
- **CLI:** Node.js, TypeScript
- **Analysis:** Slither, custom detectors
- **Frontend (dashboard):** Next.js, Tailwind CSS
- **Testing:** Foundry integration
